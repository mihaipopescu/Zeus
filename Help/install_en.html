Zeus installation

The installation is made through the"setup.exe" program or "Zeus Setup.msi".
Follow the instructions which appear and Zeus will be installed in the system in a transparent way.
The Zeus application consists in two libraries (zeus.dll and zipaes.dll) which are registered with Windows Explorer.
The system must be restarted so that the program can be successfully installed.
Zeus appears in the Start menu and has only one reference to the help files.

System requirements

Zeus works on Windows 2000/XP/Vista but you should have administrator rights when you install it. If the overlay icon doesn't appear restart the computer or just the "explorer.exe" (from "Task Manager").
