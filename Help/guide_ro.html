Cine este Zeus ?

Zeus este acronimul pentru Zip Encryption Unified Shell extension.

Intr-o fraza, Zeus este o aplicatie care se integreaza cu Windows Explorer TM si care asigura protejarea prin criptare si compresie a fisierelor intr-un format deschis si anume formatul ZIP la care se adauga noile extensii de criptare AES formulate de WinZIPTM. Fisierele protejate sunt accesibile si prin deschiderea cu WinZIP sau orice alt software utilitar care implementeaza extensia de criptare ca 7-zip.
