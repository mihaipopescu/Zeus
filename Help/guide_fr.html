Qui est Zeus ?

Le Zeus est l'acronyme pour le Zip Encryption Unified Shell extension.

Dans une expression, le Zeus est une application qui intègre avec Windows Explorer et qui assure la protection par le chiffrage et la compression des chemises dans un format ouvert à savoir le format ZIP auquel c'est supplémentaire nouvelle AES prolongation de chiffrage formulée par WinZIPTM. Les dossiers protégés sont accesible également par l'ouverture avec le WinZIP ou avec un autre logiciel utilitaire qui met en application la prolongation de chiffrage comme 7-zip.
